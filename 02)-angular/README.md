## inatalalación
- npm install -g @angular/cli
## crear proyecto 
- ng new bases --no-standalone
## inciar serve
- ng serve -o

- ng g m moduleName
## generar un componente
- ng g c moduleName/components/componentName --skip-tests --inline-style

### build
- ng build
